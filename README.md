# 💌 Valentine's Day - Interactive Proposal Web App

Create a romantic and interactive web application to ask that special someone to be your Valentine! Features a beautiful letter opening animation, playful interactions, and email notifications when they say "Yes!" 💕

## ✨ Features

- 🎁 **Letter Opening Animation** - Beautiful animated envelope
- 💖 **Personalized Messages** - Uses your partner's name throughout
- 📧 **Email Notifications** - Get notified when they say yes!
- 🎨 **Floating Hearts** - Infinite romantic animations
- 📱 **Mobile Responsive** - Works on all devices
